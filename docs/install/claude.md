# Installing on Claude

## Manual Installation

1. Navigate to your **Project Settings** or **System Prompt** configuration in Claude.
2. Copy the contents of the relevant skills you wish to enforce (from `skills/ps-*/SKILL.md`).
3. specific instructions from [AI Prompt Wrapper](../ai-prompt-wrapper.md) into your project's custom instructions or system prompt.

## Recommended Setup

Create a `CLAUDE.md` or `.prompt.md` file in your project root containing the core principles and skill definitions so they are always available in your context.
